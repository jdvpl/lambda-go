## creating a lambda

### creating go.mo

```
go mod init go.mod
```

### getting a custom package

```
go get github.com/aws/aws-lambda-go/lambda
go get github.com/aws/aws-lambda-go/events
```

## instaling aws go

```
go get github.com/aws/aws-sdk-go-v2/aws
go get github.com/aws/aws-sdk-go-v2/config
```
